作品名： Animation Tool
制作期間：２ヶ月

この作品の目的は 2 つあります。まず、FBX ファイルの構造や使用方法を理解することです。
そして、ゲームのアニメーションを管理するためのツールを自作し、より滑らかなゲームプレイを実現することです。


勉強の目的でライブラリを使わずに、モデルやアニメーションの FBX ファイル読込機能を
自作しました。ツール内では、メッシュ、モデル、アニメーションのデータを最適化する
おかげで、ゲーム起動時の負荷を軽減し、実行速度を向上させます。

通常機能では再生・停止、ループや再生速度の調整ができます。
-ブレンドの速さはアニメーションの切り替えフレーム数を設定できます。
-次のアニメーションの機能はアニメーションがループしていない場合、アニメーションが
終了したら、選択したアニメーションを自動的に再生します。
-最後に非表示機能ではモデルとその子どもを非表示できます。例を挙げれば、鎧や武器を
持ったキャラクターなど、レベルに応じてパーツを隠したり、表示したりすることが
できて、戦闘中の部位破壊もできます。
