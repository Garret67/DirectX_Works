作品名：森の守護者
制作期間：2週間

この作品の目的は、私のアニメーションツールの機能性を示し、それがゲーム内でどの
ように活用されるかを示すことです。特に処理速度に重点を置いています。


FBX ファイルの読み込みやデータ整理、バッファーデータなどの処理はすべてアニメーシ
ョンツールで行われるおかげで、特に初期化処理の速度が向上しました（ゲームはそれらの
情報をまとめたバイナリファイルを読み込んでいます）。

プレイヤーの速度に応じてアニメーションの速度を調整する（ジョイスティック）。
プレイヤーがクリティカル攻撃を行うためにボタンを押すタイミングを設定するために、
アニメーションのキーフレームが使用されています。

このゲームのマップでは、プレイヤーの移動に応じて自動的に再配置されます。
これにより、広大で多くの敵が存在するかのような感覚を演出しました。この効果は、
マップが常に再利用される 4 つの等しい部分に分割されているために実現されます。
そのため、リソースの利用が低く、プレイヤーが移動するたびにマップを生成するよりも
処理が速いです
